# Echo Maze

Echo Maze is a minimalist exploration puzzle game where you navigate a completely dark maze using sound pulses. Emit echo waves to briefly reveal the walls around you, avoid enemies attracted to sound, and find the glowing exit to advance.

## Tech Stack

- HTML
- CSS
- JavaScript
- Canvas API

## Running the Game

Open `index.html` in a modern browser (Chrome, Firefox, Edge, Safari). For the best experience during development, serve the folder with a simple static HTTP server so that module imports and future assets work correctly (for example, using `npx serve` or any local web server).

## Controls

- **Desktop**
  - Move: `WASD` or Arrow Keys
  - Echo: `Space`
- **Mobile**
  - Virtual joystick to move
  - Echo button to emit a pulse
